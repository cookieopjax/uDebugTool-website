Release 1.0 版本

2020年3月10日
  1. 移除在沒有輸出時的警告視窗
  2. 新增功能：按確定後會直接將輸出複製到剪貼簿上
  3. 新增功能：選擇檔案指定只可為執行檔（*.exe）類型
  4. 新增功能：會記住上次使用的檔案位置
  5. 新增功能：新增 uDebug.com 的連結
  6. Bug 修復


Alpha 0.2.2 版本

2020年2月5日
  1. 更新軟體字型與調整界面排版
  2. 修改安裝包圖示
  3. 新增功能：防止使用者在未選擇執行檔路徑就開始輸出
  4. 新增功能：若程式沒有輸出則會彈出通知視窗


Alpha 0.1.1 版本

2020年2月4日
  1. 將軟體運行文件封裝成安裝包
  2. 軟體安裝完成後會自動產生捷徑


Alpha 0.1.0 版本

2020年2月3日
  1. 使用 Qt 與 C++ 開發圖形使用者介面
  2. 新增功能：使用者直接選取執行檔
  3. 新增功能：輸出結果會立即顯示在軟體中


Build 初始版本

2020年2月2日

以 C 語言在 Visual Studio 中開發，為小黑窗程式，尚未建立圖形化介面

  1. 新增功能：手動輸入來源資料夾位置與執行檔案名稱
  2. 新增功能：使用者可以直接在軟體中輸入測資
